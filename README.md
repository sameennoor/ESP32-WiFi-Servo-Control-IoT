# ESP32 Wi-Fi Controlled Servo Motor (0°–180° Rotation)
## Introduction
This project demonstrates an IoT-based embedded system where an ESP32 microcontroller is used to control a servo motor over a Wi-Fi network. The servo motor can rotate precisely between 0° and 180° using PWM signals generated by the ESP32. This system is a practical example of combining wireless communication with hardware control for IoT applications.

## Objective
The objective of this project is to design a Wi-Fi enabled servo control system using ESP32 that allows remote operation of a servo motor. The system focuses on real-time control, automation, and demonstration of IoT-based device communication.

## Hardware Requirements
- ESP32 Development Board
- Servo Motor (SG90 or similar)
- Jumper Wires
- External Power Supply (recommended for servo)
## Software Requirements
- Arduino IDE
- ESP32 Board Package
- WiFi Library
- Servo Library (ESP32 compatible)

## Circuit Connection
- Servo Signal Pin → ESP32 GPIO (e.g., GPIO 18 / GPIO 2)
- Servo VCC → 5V External Supply
- Servo GND → Common Ground with ESP32

## Working Process
1. ESP32 connects to Wi-Fi network
2. User sends command (angle value) via web interface
3. ESP32 receives and processes the command
4. PWM signal generated on GPIO pin
5. Servo motor rotates to required angle (0°–180°)

## Project Purpose
This project demonstrates the integration of IoT, wireless networking, and embedded systems. It highlights how ESP32 can be used for remote hardware control, making it suitable for smart automation, robotics, and industrial IoT applications.

## Conclusion
The ESP32 Wi-Fi Controlled Servo System successfully demonstrates remote hardware control using wireless communication. It provides a strong foundation for IoT-based automation systems and can be extended for advanced robotics and smart device applications.
